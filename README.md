FitTrack 🚀

GitHub stars · GitHub issues · GitHub license

FitTrack is an iOS fitness app designed to help users track workouts, monitor progress, and achieve their fitness goals. Log exercises, view progress charts, set reminders, and create custom workouts — all in one sleek app.

⚡ Features

Log workouts with exercises, sets, reps, and duration
Track progress with charts and summaries
Set personalized fitness goals
Activity reminders to stay consistent
Create and save custom workouts
Sync data with Apple Health via HealthKit

🛠️ Tech Stack

Language: Swift
Framework: SwiftUI / UIKit
Database: CoreData / Realm
iOS SDK: iOS 16+
APIs: HealthKit
Dev Tools: Xcode, Git

🚀 Getting Started
1️⃣ Clone the repo:

git clone https://github.com/yourusername/FitTrack-iOS.git
cd FitTrack-iOS

2️⃣ Open FitTrack.xcodeproj in Xcode
3️⃣ Select your target device or simulator
4️⃣ Build and run (⌘R)
